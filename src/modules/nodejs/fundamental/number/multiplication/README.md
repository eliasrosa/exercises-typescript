# Exercícios

## Double

Crie uma função chamada `double(x)` que multipla o valor de `x` por `2`.

Exemplo:

```TypeScript
double(0) => 0
double(5) => 10
double(10) => 20
```

## Triple

Crie uma função chamada `triple(x)` que multipla o valor de `x` por `3`.

Exemplo:

```TypeScript
triple(0) => 0
triple(5) => 15
triple(10) => 30
```

## Multiple

Crie uma função chamada `multiple(x, y)` que multipla o valor de `x` por `y`.

Exemplo:

```TypeScript
triple(0, 2) => 0
triple(5, 10) => 50
triple(10, 10) => 100
```
